# Dimensionality Reduction

Included data for dimensionality reduction:
- **mnist_784.zip**: MNIST-784 dataset for SOM visualization and analysis. Data in zip because it has over 100MB which is limit for files in github.
- **dataset.arff**: Bike Sharing Rental dataset in ARFF format, used for PCA and t-SNE analysis.

Comparing linear (PCA) and non-linear (t-SNE) and analysis of SOM visualization in **Nonlinear_Dimensionality_Reduction.ipynb**. The markdown cells contain answers to the questions.
